how to track a file: git add name of the file.

how to untrack a file: git rm --cached name of the file.

how to restore a deleted file: git restore name of the file.

how to clear the command prompt: clear.

how to track all files: git add --all or git add . or git -A

to commit means to take a snapshot of your repository at a point in time.

to view commit history: git log 

to view a short version of commit history: git log --oneline

to create a new branch: git branch name of the branch

to switch branches: git switch name of the branch

to merge changes in two braches: git merge -m "some message" branch name.
